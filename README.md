## Tech Stack

Hype is built with cutting-edge technologies:

- **Language**: Typescript.
- **Frontend**: React, TanStack, Astro, TailwindCSS.
- **Backend**: Node, Inngest, Resend.
- **Database**: Neon Postgres, Drizzle.
- **Auth**: Better Auth.
- **Tooling**: Bun, Turborepo, Vite, Biome.
